ΚΡΑΤΗΣΕΙΣ ΔΩΜΑΤΙΩΝ - ΟΔΗΓΙΕΣ iPHONE

Τι περιέχει:
- index.html  -> η εφαρμογή
- manifest.json, sw.js, icon-192.png, icon-512.png -> απαραίτητα αρχεία για να μπει σαν app στην οθόνη αφετηρίας

Τι άλλαξε:
- Ο μήνας φαίνεται επάνω στο ημερολόγιο
- Σε κάθε ημέρα φαίνεται καθαρά η ημερομηνία της ημέρας
- 2 δωμάτια: Μικρό δωμάτιο / Μεγάλο δωμάτιο
- Επιλογή πολλών ημερών για μία κράτηση
- Έλεγχος διπλοκράτησης

Σημαντικό:
Για να το βάλεις σαν app στο iPhone, πρέπει πρώτα να ανοίγει από Safari ως ιστοσελίδα.
Μετά πατάς Κοινή χρήση > Προσθήκη στην οθόνη Αφετηρίας > Open as Web App > Προσθήκη.
